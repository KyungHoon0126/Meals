# Meals_Program
&#127857; C# & WPF로 제작한 대구 소프트웨어 고등학교 학교급식정보 프로그램입니다.

> NEIS API를 이용하여 대구 소프트웨어 고등학교의 급식 정보를 불러온 후, 데이터를 파싱하였습니다. 

## Function
- 전날 아침, 점심, 저녁 급식 정보 제공
- 당일 아침, 점심, 저녁 급식 정보 제공
- 다음날 아침, 점심, 저녁 급식 정보 제공
